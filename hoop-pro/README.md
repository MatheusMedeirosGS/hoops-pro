Site Institucional – Hoops Pro - Produtos e Serviços Relacionados Ao Mundo Hooper

O projeto consiste no desenvolvimento de um site institucional para a divulgar produtos e serviços especializados no mundo do basquete, o objetivo é auxiliar esse esporte, reforçar credibilidade no mercado e facilitar o contato entre clientes e a empresa, utilizando HTML, CSS e Bootstrap para criar uma página moderna, responsiva e alinhada à identidade visual da empresa.
